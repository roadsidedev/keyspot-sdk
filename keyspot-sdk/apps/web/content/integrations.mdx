# Framework Integrations

Framework wrappers are available from `@roadsidelab/keyspot-sdk/frameworks`:

```typescript
import { withKeySpot, wrapAnthropic, wrapOpenAI } from '@roadsidelab/keyspot-sdk/frameworks';
```

## LangChain

```typescript
const guardedChain = withKeySpot(myChain, guard);
```

## Anthropic

```typescript
const client = wrapAnthropic(anthropic, guard);
```

## OpenAI

```typescript
const client = wrapOpenAI(openai, guard);
```

## Self-Hosted Server

For self-hosted deployments with Express middleware, use the Docker image:

```bash
docker build -t keyspot .
docker run -p 3000:3000 keyspot
```
