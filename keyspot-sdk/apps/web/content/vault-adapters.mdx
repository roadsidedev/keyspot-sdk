# Vault Adapters

All vault adapters are available from the main `@roadsidelab/keyspot-sdk` package:

```typescript
import { InMemoryVaultAdapter, EnvVaultAdapter, DotEnvVaultAdapter } from '@roadsidelab/keyspot-sdk';
```

## In-Memory (default for testing)

```typescript
const vault = new InMemoryVaultAdapter();
```

## Environment Variables

```typescript
const vault = new EnvVaultAdapter({
  prefix: 'KEYSPOT_'
});
```

## Dotenv Files

```typescript
const vault = new DotEnvVaultAdapter({
  path: '.env.vault'
});
```

## HashiCorp Vault

```typescript
import { HashiCorpVaultAdapter } from '@roadsidelab/keyspot-sdk';

const vault = new HashiCorpVaultAdapter({
  address: 'https://vault.company.com',
  token: process.env.VAULT_TOKEN,
  namespace: 'keyspot'
});
```

## AWS Secrets Manager

```typescript
import { AWSSecretsAdapter } from '@roadsidelab/keyspot-sdk';

const vault = new AWSSecretsAdapter({
  region: 'us-east-1',
  prefix: 'keyspot/'
});
```
